# GMPEhaz
A R package for wrapping GMPE functions from Norman Abrahamson's fortran [HAZ](https://github.com/abrahamson/HAZ) code.

# Install

```{r}
install.packages("devtools")
devtools::install_github("personlin/GMPEhaz")
```
